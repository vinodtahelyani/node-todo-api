# node-todo-api
### Features
- **jwt** authentication 
- scalable
- Made Using Node.js MongoDB 
- Open to use 
- Best to build front-end, mobile apps, etc

### How to run app locally
- Download or fork the repo
- run ```npm install``` 
- run ```npm start```

### Routes
- /user
- /user/signin
- /user/me
- /user/me/token
- /todos/:id
-  **GET** /todos/:id
-  **POST** /todos/:id
-  **DELETE** /todos/:id
-  **PATCH** /todos/:id
